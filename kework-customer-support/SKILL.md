# FusionAIスクール カスタマーサポート

FusionAIスクールの受講生・見込み客からの問い合わせに対し、回答文面のドラフトを作成するスキル。

## 重要な前提

- 金額・期限・個別条件など具体的な数値は、必ず谷内に確認してからお客様に送ること
- このスキルが出力するのは「ドラフト + 確認すべきポイント」であり、そのまま送信してはいけない
- お客様への回答は丁寧語（です・ます調）で統一する

## コース情報

### ビギナーコース
- 対象: AI初心者、プログラミング未経験者
- 内容: AI基礎、プロンプトエンジニアリング入門、実践ワーク

### スタンダードコース
- 対象: ビギナー修了者、AI活用経験者
- 内容: 応用的なAI活用、業務自動化、プロジェクト実践

### コースアップグレード（ビギナー → スタンダード）
- ビギナーコース修了後にスタンダードコースへアップグレード可能
- 差額についての具体的な金額は谷内に確認が必要
- アップグレード条件・時期についても谷内に確認が必要

## 支払い方法

- **銀行振込**: 振込先口座情報は谷内に確認して案内する。振込期限は個別に設定される場合があるため都度確認
- **クレジットカード**: 対応可否・決済方法の詳細は谷内に確認

## 標準回答テンプレート

### 領収書発行
- 領収書の発行は対応可能
- 宛名・但し書きの希望を確認する
- 発行形式（PDF / 郵送）は谷内に確認が必要

**確認ポイント**: 宛名、但し書き、発行形式、発行タイミング

### クーリングオフ
- 特定商取引法に基づくクーリングオフ制度が適用される場合がある
- 契約書面受領日から8日以内が一般的な期間
- 具体的な適用条件・手続きは谷内に確認が必要

**確認ポイント**: 契約日、契約書面の受領日、適用可否の法的判断

### 中途解約
- 特定継続的役務提供に該当する場合、中途解約の権利がある
- 解約時の精算方法（受講済み分の計算、違約金の有無）は谷内に確認が必要
- 返金がある場合の振込先情報を確認する

**確認ポイント**: 契約内容、受講進捗、精算金額、違約金の有無、返金振込先

### コースアップグレード
- ビギナーコースからスタンダードコースへの変更は可能
- 差額・手続き方法・開始時期は谷内に確認が必要

**確認ポイント**: 現在の受講進捗、差額、アップグレード開始時期

### サポート範囲
- 質問サポートの範囲はコースごとに異なる
- 対応チャネル（メール、チャット等）と対応時間を案内する
- コース外の質問への対応可否は谷内に確認

**確認ポイント**: 受講中のコース、サポートチャネル、対応時間帯、対応範囲の境界

### 支払い方法の案内
- 銀行振込またはクレジットカードから選択可能
- 銀行振込の場合: 振込先口座・期限を案内
- クレジットカードの場合: 決済手順を案内

**確認ポイント**: 振込先口座情報、振込期限、クレジットカード決済の具体的手順

## 出力フォーマット

回答を作成する際は、以下の2部構成で出力すること。

### 1. 回答文面ドラフト

お客様にそのまま送れる形式の丁寧な文面。冒頭に宛名、末尾に署名欄を含める。

```
{お客様名} 様

お問い合わせいただきありがとうございます。
FusionAIスクールの{担当者名}です。

{本文}

ご不明な点がございましたら、お気軽にお問い合わせください。

FusionAIスクール
{担当者名}
```

### 2. 谷内への確認ポイント

ドラフト内で確定できなかった項目をリストアップする。

```
【谷内確認事項】
- [ ] {確認が必要な項目1}
- [ ] {確認が必要な項目2}
```

## 対応フロー

1. 問い合わせ内容を分類する（領収書 / 解約 / アップグレード / 支払い / サポート範囲 / その他）
2. 該当する標準回答テンプレートを参照してドラフトを作成する
3. 確認が必要なポイントを洗い出す
4. 回答文面ドラフト + 確認ポイントを出力する
5. 谷内の確認・修正を経てからお客様に送信する
