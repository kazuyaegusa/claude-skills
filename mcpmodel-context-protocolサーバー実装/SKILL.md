# MCP（Model Context Protocol）サーバー実装

> ダウンローダーをMCPサーバーとして公開し、Claude等のAIエージェントからツールとして呼び出し可能にする

- 出典: https://github.com/qiye45/wechatDownload
- 投稿者: qiye45
- カテゴリ: claude-code-workflow

## なぜ使うのか

AIエージェントがリサーチタスクの一環でWeChat記事を自動取得・分析できるようにするため。手動ダウンロード→AIに渡す、ではなくAIが自律的にコンテンツ取得できる

## いつ使うのか

AIエージェントワークフロー内でWeChat記事の自動収集・分析を組み込みたい場合

## やり方

1. MCPサーバー仕様に従ってツールエンドポイントを実装
2. 提供機能：
   - 記事ダウンロード（単体・複数・合集）
   - 公式アカウントID取得
   - 記事データのCSVエクスポート
3. MCPクライアント（Claude Code等）から呼び出し可能にする
4. Skill形式でも同等の機能を提供（SKILL.md定義）

### 入力

- 記事URL
- 公式アカウントID
- ダウンロード条件（日付範囲・閲覧数閾値等）

### 出力

- 記事ファイル（指定形式）
- CSVデータ（メタデータ）

## 使うツール・ライブラリ

- MCP SDK
- Skill定義（Claude Code）

## 前提知識

- WeChatアカウント（デスクトップ版またはモバイル版）
- HTTP通信の基本知識（トークン・認証ヘッダーの概念）
- MCP/Skillの基本概念（AIエージェントから外部ツールを呼び出す仕組み）
- Python環境（MCP実装を自前でカスタマイズする場合）
