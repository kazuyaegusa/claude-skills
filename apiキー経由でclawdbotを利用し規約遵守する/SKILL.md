# APIキー経由でClawdBotを利用し、規約遵守する

> Anthropic認証の意図せぬ利用を避けるため、Claude API経由でClawdBotを運用し、利用規約を確認した上で自己責任で使用する

- 出典: https://x.com/ai_masaou/status/2014624157112402078
- 投稿者: まさお@AI駆動開発
- カテゴリ: other

## なぜ使うのか

2026年1月上旬にAnthropicがClaude認証の規制を強化し、OpenCodeやClawdBotが影響を受けたため、APIキー経由が推奨される運用方法となったため

## いつ使うのか

Anthropic認証の規約違反を避け、安定してClawdBotを運用したい場合

### 具体的な適用場面

- Discord/Slackでチーム作業中にAI支援が必要な場面
- 複数日にまたがるプロジェクトで文脈を引き継ぎたい場合
- ハンズフリーで音声指示を出しながらPC作業を進めたい場面
- 既存のチャットワークフローにAI機能を統合したい組織
- ブラウザ自動操作・情報収集・フォーム入力を定期的に行う業務

## やり方

1. Anthropic ConsoleでAPIキーを発行 2. ClawdBot設定ファイルに `ANTHROPIC_API_KEY` を記載(または環境変数として設定) 3. オンボーディング時にAPIキー認証を選択 4. 利用規約を読み、自己責任で運用することを理解 5. セキュリティ管理(APIキーの漏洩防止)を徹底

### 入力

- Anthropic APIキー
- APIキーを保存する設定ファイルまたは環境変数

### 出力

- 規約遵守した安定運用
- API経由のClaude利用

## 使うツール・ライブラリ

- Anthropic API
- ClawdBot

## 前提知識

- Node.js/npm実行環境
- Discord/Slack/TelegramのBot作成・トークン取得方法
- Anthropic APIキーの取得方法
- 基本的なコマンドライン操作
- Claude Codeまたは同等のAIエージェントツールの概念理解
