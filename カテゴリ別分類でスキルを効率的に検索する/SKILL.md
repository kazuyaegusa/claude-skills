# カテゴリ別分類でスキルを効率的に検索する

> 10の標準カテゴリ（Document Skills, Development & Code Tools, Data & Analysis等）を使ってスキルを素早く発見する

- 出典: https://github.com/BehiSecc/awesome-claude-skills
- 投稿者: BehiSecc
- カテゴリ: other

## なぜ使うのか

150以上のスキルから目的のものを見つけるには、体系的な分類が不可欠。カテゴリはユースケース指向で設計されており、直感的に探せる

## いつ使うのか

タスクの種類（文書作成、データ分析、Web開発等）が明確な場合。カテゴリ名から目的のスキル群に素早く到達できる

## やり方

1. README.mdの目次から関心のあるカテゴリ（📄 Document Skills, 🛠 Development & Code Tools等）を選択
2. カテゴリ内のスキル一覧を概要説明とともにスキャン
3. 気になるスキルのGitHubリンクをクリックして詳細確認
4. 複数カテゴリにまたがる場合は横断検索（例: セキュリティ監査なら🛡 Security + 🛠 Development）

### 入力

- タスクの種類・ドメイン

### 出力

- 候補スキルのショートリスト

## 使うツール・ライブラリ

- GitHub README
- Markdown目次

## 前提知識

- Claude Codeの基本的な使い方（スキルのロード・呼び出し方法）
- GitHubの基本操作（リポジトリのクローン、ファイル取得）
- Markdownの読み書き
- ~/.claude/skills/ディレクトリの存在と役割の理解

> 「OpenPaw - 38-skill bundle that turns Claude Code into a personal assistant.」（コレクション例）
