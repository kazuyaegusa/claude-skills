# 言語・プラットフォーム絞り込み

> アイコン（🐍 Python, 📇 TypeScript, 🏎️ Go, 🦀 Rust等）とプラットフォーム表示（☁️ Cloud, 🏠 Local, 🍎 macOS等）で実装技術を確認する

- 出典: https://github.com/punkpeye/awesome-mcp-servers
- 投稿者: punkpeye
- カテゴリ: dev-tool

## なぜ使うのか

プロジェクトで使用する言語や実行環境に合わないServerは導入できないため、事前にフィルタリングする必要がある

## いつ使うのか

特定の技術スタックや実行環境でMCP Serverを構築する必要がある時

## やり方

1. リスト内の各ServerエントリでアイコンとOS記号を確認
2. 例: macOS専用ツールなら🍎、Python実装なら🐍
3. 自分の環境（例: Linux 🐧 + TypeScript 📇）に一致するものを選ぶ

### 入力

- 開発言語の選定（Python/TypeScript/Go等）
- 実行環境（Local/Cloud/特定OS）

### 出力

- 技術的に適合するMCP Serverの候補リスト

## 使うツール・ライブラリ

- 各言語のSDK（例: FastMCP for Python/TypeScript）

## 前提知識

- Model Context Protocol (MCP)の基本概念（Server/Client/Tools/Resources）
- Claude DesktopまたはMCP対応クライアントの使用経験
- GitHubの基本操作（リポジトリのクローン、README閲覧）
- JSON設定ファイルの編集方法

## 根拠

> 🐍 – Python codebase, 📇 – TypeScript (or JavaScript) codebase, 🏎️ – Go codebase, 🦀 – Rust codebase
