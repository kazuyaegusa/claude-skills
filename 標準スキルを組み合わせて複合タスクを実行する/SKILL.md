# 標準スキルを組み合わせて複合タスクを実行する

> ClawdBotに標準搭載されているブラウザ自動操作、Whisper音声認識、nanobanana、FFmpeg等のスキルを選択・組み合わせて、複雑な業務タスクをチャット指示のみで完了させる

- 出典: https://x.com/ai_masaou/status/2014624157112402078
- 投稿者: まさお@AI駆動開発
- カテゴリ: skill-management

## なぜ使うのか

個別ツールを手動で連携させる手間を省き、チャットからの自然言語指示だけで複数ツールを横断した処理を実行できるため

## いつ使うのか

情報収集→加工→保存のような複数ステップの定型業務を自動化したい場合

### 具体的な適用場面

- Discord/Slackでチーム作業中にAI支援が必要な場面
- 複数日にまたがるプロジェクトで文脈を引き継ぎたい場合
- ハンズフリーで音声指示を出しながらPC作業を進めたい場面
- 既存のチャットワークフローにAI機能を統合したい組織
- ブラウザ自動操作・情報収集・フォーム入力を定期的に行う業務

## やり方

1. ClawdBotインストール時のオンボーディングで必要なスキル(ブラウザ操作、Whisper、FFmpeg、Spotify、Obsidian等)を選択 2. チャットで「WebサイトXから情報を取得してスライド作成」「音声ファイルを文字起こししてObsidianに保存」等の複合指示を出す 3. ClawdBotが各スキルを自動連携して実行 4. 必要に応じて追加スキルを後から導入

### 入力

- チャットでの自然言語指示
- 各スキルが必要とする入力(URL、音声ファイル、動画ファイル等)

### 出力

- ブラウザから取得した情報
- 文字起こしテキスト
- 編集済み動画
- 自動生成スライド
- Obsidian/Reminderへの保存結果

## 使うツール・ライブラリ

- ClawdBot標準スキル群
- Whisper
- nanobanana
- FFmpeg
- Spotify API
- Apple Reminder/Note
- Obsidian

## 前提知識

- Node.js/npm実行環境
- Discord/Slack/TelegramのBot作成・トークン取得方法
- Anthropic APIキーの取得方法
- 基本的なコマンドライン操作
- Claude Codeまたは同等のAIエージェントツールの概念理解

> 「npmコマンド一発でインストール完了」
