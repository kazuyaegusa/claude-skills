# Backend Tool Renderingでツール実行結果をUI化

> バックエンドのエージェントツールがReact/AngularコンポーネントをレスポンスとしてクライアントUIにレンダリングさせる

- 出典: https://github.com/CopilotKit/CopilotKit
- 投稿者: CopilotKit
- カテゴリ: agent-orchestration

## なぜ使うのか

エージェントのツール実行結果をテキストで返すだけではユーザーが情報を操作・確認しにくい。UIコンポーネントとして返すことでインタラクティブな体験を提供できる

## いつ使うのか

検索結果・グラフ・フォーム・カレンダーなどエージェントの出力をリッチUIで表現したい場合

## やり方

1. バックエンドツール定義でUIコンポーネントを返す仕様を記述
2. エージェントがツールを呼び出し、レスポンスにUI定義を含める
3. フロントエンドのCopilotKitがレスポンスを受信し、対応するコンポーネントを動的レンダリング
4. ユーザーがUIを操作すると、その入力がエージェントに戻る

### 入力

- バックエンドツール定義
- UIコンポーネントの仕様（AG-UIプロトコル）

### 出力

- クライアントにレンダリングされたReact/Angularコンポーネント

## 使うツール・ライブラリ

- CopilotKit
- AG-UIプロトコル

## 前提知識

- React/Angularの基礎知識
- Hooksの理解（useStateなど）
- エージェント・LLMアプリケーションの基本概念
- 非同期処理・ストリーミングの理解
