# cron自動化とプラットフォーム配信

> 自然言語でcronスケジュールを設定し、結果を任意のメッセージングプラットフォームに配信

- 出典: https://github.com/NousResearch/hermes-agent
- 投稿者: NousResearch
- カテゴリ: agent-orchestration

## なぜ使うのか

定期レポート・バックアップ・監視タスクを手動実行するのは非効率。エージェントに自然言語で指示して自動化すれば、無人運用が可能。

## いつ使うのか

日次レポート、週次監査、夜間バックアップ等の定期タスクを自動化したい場合

## やり方

1. 自然言語でcronスケジュールを指示（例：「毎日9時にレポート生成」） 2. エージェントがcron設定を生成 3. スケジュール実行時、エージェントがタスク実行 4. 結果をTelegram/Discord等に配信

### 入力

- 自然言語のスケジュール指示

### 出力

- cron設定
- プラットフォーム配信結果

## 使うツール・ライブラリ

- hermes cron

## 前提知識

- 基本的なコマンドライン操作
- LLM APIキー（OpenRouter/OpenAI/Anthropic等のいずれか）
- gitがインストールされた環境
- Linux/macOS/WSL2のいずれか
