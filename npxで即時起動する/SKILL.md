# npxで即時起動する

> Node.jsがあれば`npx n8n`一発でローカルにn8nエディタを起動する

- 出典: https://github.com/n8n-io/n8n
- 投稿者: n8n-io
- カテゴリ: automation-pipeline

## なぜ使うのか

インストール不要でブラウザからGUIを触れるため、評価・プロトタイピングを即日始められる

## いつ使うのか

ローカルで素早く動作確認したい時、チームへのデモを準備したい時

## やり方

1. Node.jsをインストール
2. `npx n8n`を実行
3. http://localhost:5678 をブラウザで開く
4. GUIでワークフローを作成

### 入力

- Node.js環境

### 出力

- http://localhost:5678 で動くn8nエディタ

## 使うツール・ライブラリ

- Node.js
- npx

## コード例

```
npx n8n
```

## 前提知識

- Node.js（npx起動の場合）またはDocker（コンテナ起動の場合）の基本操作
- ワークフロー自動化の基本概念（トリガー、アクション、条件分岐）
- AIエージェント機能を使う場合はLangChainの基礎知識
