# KEWORK Content Writer

FusionAIスクールおよびAIXEED関連のマーケティング・教育コンテンツを作成・更新するスキル。

## タスク判別

入力内容から以下のいずれかのワークフローを選択する。

| キーワード | ワークフロー |
|---|---|
| LP、ランディングページ、案内資料 | LP テキスト更新 |
| LINE、配信、要約 | LINE メッセージ整形 |
| 説明会、要点、まとめ | コンテンツ要約 |
| Webサイト、確認、プロモーション動画 | Webサイトレビュー & 要約 |
| アンケート、自動化、コード、GAS | GAS スクリプト作成 |
| 動画講義、撮り直し、リニューアル | 動画コンテンツ更新計画 |

---

## 1. LP テキスト更新ワークフロー

説明会やミーティングの内容をもとに、LP・案内資料のコピーを更新する。

### 手順

1. **素材の確認**: 説明会の議事録・録画書き起こし・共有資料を読み込む
2. **キーポイント抽出**: 以下の観点で要点を抜き出す
   - 新しいカリキュラム内容・変更点
   - 受講生の成果事例・数値実績
   - 料金・特典の変更
   - 講師陣の追加・変更
   - 差別化ポイント（他スクールとの違い）
3. **既存LPとの差分確認**: 現行LPテキストと抽出した要点を比較し、更新が必要な箇所を特定
4. **コピー作成**: 更新箇所ごとに新しいコピーを作成

### 出力フォーマット

```
## LP更新提案

### 更新箇所一覧
| セクション | 現行 | 更新案 | 更新理由 |
|---|---|---|---|

### ヘッドコピー案（3案）
1. ...
2. ...
3. ...

### 各セクション更新テキスト

#### [セクション名]
**現行:**
> （現在のテキスト）

**更新案:**
> （新しいテキスト）

**変更理由:** ...
```

## 2. LINE メッセージ整形

説明会内容やお知らせをLINE公式アカウント配信用に整形する。

### LINE配信の制約・ルール

- **文字数**: 1メッセージあたり500文字以内を目安（長文は分割）
- **冒頭**: 開封を促すフック文（1行目で興味を引く）
- **絵文字**: 視認性のため見出しや区切りに適度に使用（乱用しない、1メッセージに3〜5個程度）
- **CTA（行動喚起）**: メッセージ末尾に1つ配置。リンクはURLを短縮して記載
- **改行**: 3〜4行ごとに空行を入れて読みやすくする
- **トーン**: 親しみやすく、かつ専門性を感じさせる

### 出力フォーマット

```
## LINE配信メッセージ

### メッセージ1（○○文字）
---
[メッセージ本文]
---

### メッセージ2（分割が必要な場合）
---
[メッセージ本文]
---

### 配信タイミング提案
- 推奨日時: ...
- 理由: ...
```

### 手順

## 注意事項

- FusionAIスクールとAIXEEDのブランドガイドラインに従う
- 受講生の個人情報や未公開の料金情報は出力に含めない
- LP更新案は「提案」として提示し、最終判断は谷内さんまたは担当者が行う
- 数値実績を記載する場合は出典を明記する
