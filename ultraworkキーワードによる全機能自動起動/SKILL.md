# `ultrawork`キーワードによる全機能自動起動

> プロンプトに`ultrawork`（または短縮形`ulw`）を含めるだけで、全ての並列エージェント・探索・継続強制が自動的に発動する「魔法のキーワード」パターン

- 出典: https://github.com/code-yeongyu/oh-my-openagent
- 投稿者: code-yeongyu
- カテゴリ: agent-orchestration

## なぜ使うのか

複雑な設定や指示を毎回書くのは面倒。キーワード一つで全機能が起動すれば、ユーザーはタスクだけに集中できる

## いつ使うのか

複雑なタスク、初めて触るコードベース、全自動でやってほしい場合

## やり方

1. プロンプトに`ultrawork`または`ulw`を含める
2. エージェントが自動的にコードベース分析・並列探索・専門エージェント起動・継続強制を実行
3. ユーザーはコーヒーを飲んで待つだけ

### 入力

- タスク記述 + `ulw`キーワード

### 出力

- 全機能が自動起動された実行

## 使うツール・ライブラリ

- oh-my-opencode

// 例:
