# MCP/Skill統合による外部呼び出し

> Claude Code等のLLMエージェントから直接このツールを呼び出し、記事ダウンロード・データ抽出を自動化する

- 出典: https://github.com/qiye45/wechatDownload
- 投稿者: qiye45
- カテゴリ: claude-code-workflow

## なぜ使うのか

手動操作なしでLLMワークフロー内に微信コンテンツ収集を組み込むことで、中国語情報の自動分析パイプラインを構築できる

## いつ使うのか

LLMエージェントに中国語のドメイン知識を与えたい時、定期的な情報収集を自動化したい時

## やり方

1. MCP（Model Context Protocol）またはSkillインターフェースとしてツールを登録
2. LLMエージェントから「公式アカウントID取得」「記事ダウンロード」「合集ダウンロード」「データエクスポート」等のコマンドを実行
3. 認証キーは事前取得済みのものを使用、または対話的に取得プロセスを実行
4. ダウンロード結果がLLMのコンテキストに返される

### 入力

- 記事URL or 公式アカウントID
- ダウンロード設定（MCP/Skillパラメータ経由）

### 出力

- ダウンロードされた記事ファイル
- メタデータ（JSON/CSV）

## 使うツール・ライブラリ

- MCP Server実装
- Claude Code Skill定義

## 前提知識

- Windows/macOS環境
- 微信デスクトップアプリのインストール
- 対象公式アカウントの記事URL
- （MCP/Skill使用時）Claude Code等のLLMエージェント環境
