# Context7 CLIでスキルをインストールする

> Context7が提供する24,000のスキルカタログから、必要なスキルをCLIコマンド1つでローカルにインストールする

- 出典: https://x.com/ai_masaou/status/2014864959864504782
- 投稿者: まさお@AI駆動開発
- カテゴリ: context-management

## なぜ使うのか

65,000リポジトリから抽出された実績あるスキルを、検索・手動コピーなしで即座に利用可能にするため

## いつ使うのか

特定のワークフロー（TDD、デバッグ、フロントエンド設計等）をAIエージェントに追加したい時

### 具体的な適用場面

- Claude CodeやCodexなどのAIエージェント環境でスキルを追加したい時
- 特定のタスク（例: TDD、デバッグ、フロントエンド設計）に特化したワークフローを導入したい時
- 他の開発者が作った高品質なスキルを探している時
- 自分のプロジェクトに適したスキルを効率的に見つけたい時

## やり方

1. Context7 CLIをインストール（具体的なコマンドは投稿に記載なし） 2. カタログから必要なスキルを検索（例: `context7 search 'TDD'`） 3. `context7 install <skill-name>` でインストール 4. インストールされたスキルがAIエージェント（Claude Code等）から自動認識される

### 入力

- Context7 CLIがインストール済み
- インストールしたいスキルの名前またはキーワード

### 出力

- ~/.claude/skills/ や同等のディレクトリにスキルファイル（SKILL.md等）が配置される
- AIエージェントがスキルを認識し、該当タスクで自動適用される

## 使うツール・ライブラリ

- Context7 CLI
- Context7 Skills（24,000スキルカタログ）

## 前提知識

- Claude CodeやCodexなどのスキル対応AIエージェント環境
- スキルとは何か（SKILL.md形式、ワークフロー定義）の理解
- CLIツールの基本的な使い方

## 根拠

> 65,000リポジトリから24,000のスキルを抽出し

> CLIコマンド1つでインストール可能になった模様
