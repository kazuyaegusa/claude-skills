# claude rc でリモートセッション起動

> `claude rc` コマンドでClaude Codeをオンラインモードで起動し、Web/デスクトップ/モバイルアプリから同セッションへの接続を可能にする

- 出典: https://x.com/hiragram/status/2026423131217555542
- 投稿者: hiragram/ひらり
- カテゴリ: claude-code-workflow

## なぜ使うのか

通常の `claude` 起動はローカル専用だが、`claude rc` はセッションをクラウド経由で公開し他デバイスからの接続を受け付ける。移動中・別端末からの操作継続が可能になる

## いつ使うのか

外出前にローカルで長時間タスクを開始し、移動中にスマホで監視・追加指示したいとき。または一時的にリモート接続したい場合

### 具体的な適用場面

- ローカルで長時間タスク（テスト・ビルド・スクレイピング等）を起動後、外出先のスマホから進捗確認・追加指示したいとき
- メインPCで作業中にタブレット・別PCからも同じセッションに接続して引き継ぎしたいとき

## やり方

1. ターミナルで `claude rc` を実行する
2. セッションが「オンライン」モードで起動する
3. スマホ・ブラウザ等のClaudeアプリでRemote Controlのセッションに接続する
4. 接続後は通常のClaude Codeと同様に操作できる

### 入力

- ローカルマシン上のClaude Codeインストール環境
- 接続先デバイスのClaudeアプリまたはWebブラウザ

### 出力

- 複数デバイスから操作可能なClaude Codeセッション

## 使うツール・ライブラリ

- claude CLI（Claude Code）

## コード例

```
claude rc
```

## 前提知識

- Claude Codeがローカルマシンにインストール済みであること
- Anthropicのアカウント（Remote Control機能が有効なプラン）を持っていること
- 接続先デバイスでClaudeアプリまたはWebにアクセスできること

## 根拠

> `claude rc` で明示的にオンラインセッションにするほか

> `/config` で `Enable Remote Control for all sessions` をtrueにすることで常時ONにできる

> 自分のマシン上で開始したセッションにClaudeのWeb/デスクトップアプリ/モバイルアプリから繋いで操作できる
