# Copilot Tasks研究プレビュー公開

> MicrosoftがCopilot TasksをResearcher/Analystエージェントで複雑プロンプト実行できる機能として限定プレビュー公開した

- 出典: https://x.com/testingcatalog/status/2027115135911755862
- 投稿者: TestingCatalog News
- カテゴリ: agent-orchestration

## なぜ使うのか

Microsoftが一般向けコンシューマー製品にマルチエージェント協調（Researcher + Analyst）を組み込んできた。エンタープライズ向けではなく一般ユーザー向けのエージェント化が本格化している

## いつ使うのか

複数ステップの調査・分析タスクをMicrosoft製品上で自動化したいとき, Copilot/Microsoft 365環境でエージェント型ワークフローを評価・検討するとき

### 具体的な適用場面

- 複数ステップの調査・分析タスクをMicrosoft製品上で自動化したいとき
- Copilot/Microsoft 365環境でエージェント型ワークフローを評価・検討するとき

## やり方

既存のCopilot（単一の応答生成）と異なり、ResearcherとAnalystという役割特化エージェントを組み合わせて複雑なタスクを自律実行する「Tasks」機能が加わった

## 前提知識

- Microsoft Copilotアカウント
- Waitlistへの登録（限定プレビュー）

## 根拠

> Copilot Tasks can help users execute complex prompts using Researcher and Analyst agents.

> research preview for a small number of testers
