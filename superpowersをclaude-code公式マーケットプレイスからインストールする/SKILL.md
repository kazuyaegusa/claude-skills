# SuperpowersをClaude Code公式マーケットプレイスからインストールする

> Claude Code公式プラグインマーケットプレイス経由でSuperpowersを1コマンドインストールし、上記の全ワークフロースキルを自動有効化する。

- 出典: https://x.com/jiroucaigou/status/2035964407201849564
- 投稿者: 努力赚钱的菜狗
- カテゴリ: claude-code-workflow

## なぜ使うのか

個別スキルを手動設定する代わりに、Superpowersを一度インストールするだけで20以上のスキルが揃い、要件深掘り→設計→TDD実装→レビューのフロー全体が自動トリガーされるようになる。

## いつ使うのか

Claude CodeまたはCursorを使っているが、AIが生成するコードの品質・プロセスに不満があるとき

### 具体的な適用場面

- AIが生成したコードは動くが、プロジェクト拡大後に保守不能になるパターンが繰り返されているとき
- 要件が曖昧なままAIに着手させて後から仕様ズレが発覚するケースを防ぎたいとき
- 複数の独立機能を並列開発してAIの作業スループットを上げたいとき
- AIエージェントに数時間の自律作業をさせたいが品質保証が不安なとき

## やり方

方法1（推奨）: `claude plugins install superpowers` をターミナルで実行。方法2: ブラウザで https://claude.com/plugins/superpowers を開き「Install」ボタンをクリック。Cursor使用の場合は `.cursor/rules/` にスキルファイルを配置する（GitHubのREADMEに手順あり）。Codex/OpenCodeの場合は手動セットアップが必要（README参照）。インストール後は特別なコマンド不要でスキルが自動トリガーされる。

### 入力

- Claude Code CLI（インストール済み）

### 出力

- 20以上の開発ワークフロースキルの自動有効化
- 次回のコーディング依頼から自動でSuperpowersフローが適用される状態

## 使うツール・ライブラリ

- claude CLI
- Superpowers
- Claude Code公式マーケットプレイス

## コード例

```
# Claude Code経由でインストール
claude plugins install superpowers

# または公式マーケットプレイスURL
# https://claude.com/plugins/superpowers
```

## 前提知識

- Claude Code CLI または Cursor がインストール済みであること
- Gitの基本操作（branch, merge, worktree）を理解していること
- TDD（テスト駆動開発）の赤→緑サイクルの概念を知っていること
